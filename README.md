## :key: Getting Started

```bash
# clone the repo:
git clone hhttps://github.com/Regis-Oliveira/ignite-feed-class.git

# open the project:
cd Form-exercise-SH

# install the dependencies:
npm install
# or
yarn install

#execute
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
